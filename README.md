# movie-overview-website
The Movie Overview Website is a simple and dynamic platform that provides detailed information about Bollywood movies. Users can explore movie details such as title, release date, cast, genre, and ratings. The website is inspired by platforms like IMDb and Rotten Tomatoes but focuses mainly on Bollywood cinema.
